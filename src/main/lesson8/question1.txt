Опишите как работает условие if, какие существуют вариации, приведите примеры.

if обычно называют условной конструкцией (ветвлением), позволяющей выполнять тот или иной блок кода
в зависимости от выполнения условия.

1. Неполное ветвление.

if (<условие>) {
        <Блок кода>;
    }

В вариации неполного ветвления существует один блок кода, который выполняется тогда, когда условие истинно.
В противном случае (если условие ложно) никаких действий не происходит и алгоритм выполняет следующий за конструкцией код.

Пример: m, n, l - величины, определяющие длины треугольника.
Данное условие проверяет треугольник по длинам его сторон на тот факт, что треугольник равносторонний.
В случае выполнения условия выводится соответствующее сообщение.
В условии используется конъюнктивная логическая связка (сложное условие).

if (m == n && n == l) {
        alert("Треугольник равносторонний");
    }

2. Полное ветвление.

Отличается от неполного веткой else, которая запускает Блок кода 2 в том случае, когда условие ложно.

if (<условие>) {
    <Блок кода 1>;
} else {
    <Блок кода 2>;
}

Пример: следующий код делает проверку на равенство двух значений. В случае равенства выводит сообщение о равенстве величин
В остальных случаях (else) выводит сообщение о неравенстве.

if (m == n) {
    alert("m равно n");
} else {
    alert("n не равно m");
}

3. Ветвление с оператором else if.

Специальный оператор, который используется для объединения нескольких операторов if … else.
Выполняется всегда только один из блоков кода.

if(<условие 1>) {
    <Код, который будет выполнен, если условие 1 истинно>
} else if(<условие 2>) {
    <Код, который будет выполнен, если условие 1 ложно, а условие 2 истинно>
} else {
    <Код, который будет выполнен, если оба условие 1 и условие 2 ложны>
}

Пример: программа сравнивает 2 числа и выводит одно из трёх возможных сообщений.

if (m > n) {
    alert("M больше n");
} else if (m < n) {
    alert("N больше m");
} else {
    alert("n равно m");
}

4. Тернарный оператор

Тернарный оператор обеспечивает сокращенный способ записи операторов if … else.
Тернарный оператор представлен символом вопросительного знака (?) и принимает три операнда:
условие для проверки, результат для true и результат для false.

var result = (condition) ? value1 : value2

Пример: в данном примере в переменную userType будет записано значение 'Child' при истинном условии,
значение 'Adult' при ложном условии.

var age = 21;
var userType = age < 18 ? 'Child' : 'Adult';
